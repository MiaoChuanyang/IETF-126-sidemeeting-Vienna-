# IETF-126-sidemeeting-Vienna
We're planning to have a side meeting at IETF 126: Building AI Agent task based on the QoS guaranteed multi-agent collabration.

Backgroud information: The primary focus of this side meeting is to discuss, in scenarios involving multi-agent collaboration, how to create service-guaranteed tasks or task chains for users based on their intents and the network's available services and policies. This aims to achieve more efficient utilization of both network and computing resources. 

Time: Wednesday, 23 July, 2026 - AM 11:00-12:00 ( Europe/Vienna )

Location: Park Suite 4
